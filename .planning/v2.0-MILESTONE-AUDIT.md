## Defects

**DEFECT-01 (cosmetic, pre-existing):** v1 manifest handler at `api/vocab/v1/manifest.js` line 79 uses `const [from, , to] = pack.split('-')` expecting 3-part names like `german-to-norwegian`, but actual directory names are 2-part ISO codes (`de-nb`). Result: `translations[pack].to` is always `undefined` in the manifest response. Fix: `const [from, to] = pack.split('-')`.
