## Tech Debt Summary

### Phase 22: API Updates
- **Pre-existing bug:** `api/vocab/v1/manifest.js` line 79 — `translations.*.to` field is `undefined` for all 5 language packs. Destructure pattern expects long-form names (`german-to-norwegian`) but directories use ISO codes (`de-en`). Not introduced by v2.0. Fix: `const [from, to] = pack.split('-')`.
- **Documentation:** `api/vocab/README.md` lines 268/298/304 — stale audio URL examples reference `vocabulary/core/de/audio/` instead of `vocabulary/banks/de/audio/`

### Phase 23: Validation & Cleanup
