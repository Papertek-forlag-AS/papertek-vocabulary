**Part A — API grammarFeatures (INTG-01):**

In `api/vocab/v2/lookup/[language]/[wordId].js`, find line 250:
```javascript
    if (entry.comparison?.superlativ) grammarFeatures.push('grammar_superlative');
    // Check explanation text for case references (contractions, prepositions)
```

Insert between those two lines:
```javascript
    // Adjective declension — positiv key is unique to adjective declension blocks
    if (entry.declension?.positiv) {
      grammarFeatures.push('grammar_adjective_declension');
    }
```

Use `entry.declension?.positiv` (NOT `Object.keys(entry.declension).length > 0`). The `positiv` key is specific to adjective declension blocks — nouns use `cases` key, not `declension`. This avoids any false positive for non-adjective entries.

Do NOT add `grammar_adjective_genitive` — that is out of scope for INTG-01.
