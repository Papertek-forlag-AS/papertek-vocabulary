  <action>
Create `scripts/merge-banks.js` (ESM module, matching project's `"type": "module"` in package.json). The script:

1. **For each of the 8 banks** (nounbank, verbbank, adjectivebank, generalbank, articlesbank, numbersbank, phrasesbank, pronounsbank):
   - Read `vocabulary/core/de/{bank}.json` and `vocabulary/dictionary/de/{bank}.json`
   - Start with the dict bank as the base (it's the superset — all core entries exist in dict, plus dict-only entries)
   - **Deep-merge core-exclusive fields into dict entries for shared entries.** Specifically:
     - For each entry key present in BOTH core and dict (skip `_metadata`):
       - For each field in the core entry that does NOT exist in the dict entry: copy it into the merged entry
       - Do NOT overwrite any existing dict fields — core fields only fill gaps
     - Known core-exclusive fields discovered:
       - nounbank: `plural` (null for uncountable) and `genus` on 29 entries where dict is missing them
       - verbbank: `preteritum_rare` (boolean), `verb_type` (string), `preteritum_note` (string) on 83 entries
       - Other 6 banks: core is strict subset of dict, no exclusive fields (but run the merge logic anyway for safety)
   - Dict-only entries (not in core) are left unchanged (BANK-04)
   - Write `_metadata` block:
     ```json
     {
       "_metadata": {
         "source": "merged",
         "packId": "german",
         "targetLanguage": "german",
         "generatedAt": "<ISO timestamp>",
         "description": "Merged bank (core + dictionary)",
         "totalEntries": <count>,
         "curriculumEntries": <count of entries with curriculum:true>,
         "dictionaryOnlyEntries": <count of entries with curriculum:false or missing>
       }
     }
     ```
   - Sort entries by `_id` (ASCII sort, existing project convention from Phase 10 decision)
   - Write to `vocabulary/banks/de/{bank}.json` (create `vocabulary/banks/de/` directory if needed)

2. **Build core manifest** after merging all banks:
   - For each bank, collect entry IDs where `curriculum === true`
   - Write `vocabulary/banks/de/manifest.json`:
     ```json
     {
       "_metadata": {
         "packId": "german",
         "type": "core-manifest",
         "generatedAt": "<ISO timestamp>",
         "description": "Identifies curriculum entries within merged banks"
       },
       "banks": {
         "nounbank": {
           "total": 1641,
           "curriculum": 331,
           "dictionaryOnly": 1310,
           "ids": ["arbeid_paagar_noun", ...]
         },
         ...
       },
       "summary": {
         "totalWords": 3454,
         "curriculumWords": <sum>,
         "dictionaryOnlyWords": <sum>
       }
     }
     ```
   - IDs sorted alphabetically within each bank section

3. **Console output:** For each bank, log: bank name, total entries, curriculum count, dict-only count, fields merged from core. At the end, log manifest summary totals.
