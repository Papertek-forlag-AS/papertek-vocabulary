Add a `conjugations.preteritum` object to every verb entry in verbbank.json. The structure must match existing `presens`:

```json
"preteritum": {
  "former": {
    "ich": "...",
    "du": "...",
    "er/sie/es": "...",
    "wir": "...",
    "ihr": "...",
    "sie/Sie": "..."
  },
  "feature": "grammar_preteritum"
}
```

**Processing order — irregulars first (per user decision):**

1. **Strong/irregular verbs** (sein, haben, werden, gehen, kommen, fahren, finden, essen, nehmen, trinken, lesen, schreiben, sprechen, sehen, treffen, geben, helfen, schlafen, laufen, springen, schwimmen, singen, beginnen, bleiben, scheinen, fliegen, gewinnen, verlieren, vergessen, klingen, liegen, sitzen, stehen, verstehen, versprechen, bekommen). Use correct ablaut/vowel change forms. Each verb looked up individually — do NOT apply formulaic patterns.

2. **Modal verbs** (können, müssen, dürfen, sollen, wollen, mögen). Use correct modal preteritum forms (konnte, musste, durfte, sollte, wollte, mochte). Add `"verb_type": "modal"` metadata field to each modal verb entry (at the same level as `word`, `type`, etc.). For `möchten` (moechten_modal): this is the Konjunktiv II form of mögen and has no standard preteritum of its own — use `"wollte"` forms (ich wollte, du wolltest, etc.) and add `"preteritum_note": "möchten has no independent preteritum; wollte forms used as functional equivalent"`.

3. **Separable prefix verbs** (aufstehen, anfangen, einpacken, einladen, einkaufen, anrufen, aufwachen, aufräumen, fernsehen, mitnehmen, mitbringen, mitkommen, wegfahren, einschlafen, ausziehen). Store the full written form with separated prefix matching how they appear in German sentences:
   - `"ich": "stand auf"` (not "aufstand")
   - `"du": "standst auf"` (not "aufstandst")
   - Follow the presens pattern already in the file for guidance on prefix placement.

4. **Reflexive verbs** (sich treffen, sich waschen, sich duschen, sich anziehen, sich kämmen, sich schminken, sich rasieren, sich freuen, sich fühlen, sich freuen auf, sich freuen über, sich ärgern, sich ärgern über, sich interessieren, sich langweilen, sich konzentrieren, sich entschuldigen, sich verspäten, sich entspannen, sich aufregen, sich ausruhen, sich erholen, sich vorbereiten, sich setzen, sich unterhalten, sich bewegen, interessieren [refleksiv], entspannen [refleksiv]). Include reflexive pronouns in conjugation values:
   - `"ich": "... mich/mir"`, `"du": "... dich/dir"`, `"er/sie/es": "... sich"`, `"wir": "... uns"`, `"ihr": "... euch"`, `"sie/Sie": "... sich"`
   - Match the reflexive pronoun pattern from existing presens conjugations in the file.
   - For separable+reflexive verbs (sich anziehen, sich aufregen, sich ausruhen, sich vorbereiten, sich unterhalten): combine both patterns, e.g. `"ich": "zog mich an"`.

5. **Weak/regular verbs** (all remaining verbs with type "vanlig", "svakt", or untyped weak verbs). Apply the standard weak conjugation pattern:
   - ich: stem + te
   - du: stem + test
   - er/sie/es: stem + te
   - wir: stem + ten
   - ihr: stem + tet
   - sie/Sie: stem + ten
   - Handle stems ending in -t/-d (arbeiten -> arbeitete, not arbeitete without the extra -e-).
   - Handle stems ending in consonant clusters (öffnen -> öffnete).
   - STILL look up each verb individually to confirm it is actually weak, even if no `type` field exists — many verbs with type "(none)" are actually strong (e.g., gehen, trinken, finden).

6. **Verbphrases** (Rad fahren, Musik hören, Gassi gehen, fertig werden). Conjugate the verb portion only:
   - Rad fahren: `"ich": "fuhr Rad"`, `"du": "fuhrst Rad"`, etc.
   - Musik hören: `"ich": "hörte Musik"`, etc.
   - Gassi gehen: `"ich": "ging Gassi"`, etc.
   - fertig werden: `"ich": "wurde fertig"`, etc.
   - For verbphrases that are missing presens conjugations (16 verbs), add ONLY preteritum — do not add presens (that is out of scope).

**Rare-usage marker (Claude's discretion):**
For verbs whose preteritum is rarely used in spoken German (most weak verbs in everyday speech prefer Perfekt), add `"preteritum_rare": true` at the same level as `conjugations` in the verb entry. Apply this to:
- Common verbs where Perfekt strongly dominates in speech (e.g., kochen, spielen, machen, lernen, kaufen, etc.)
- Do NOT mark strong/irregular verbs (sein, haben, werden, gehen, etc.) or modals as rare — their preteritum is standard even in spoken German.

**Critical accuracy rules:**
- sein: war, warst, war, waren, wart, waren
- haben: hatte, hattest, hatte, hatten, hattet, hatten
- werden: wurde, wurdest, wurde, wurden, wurdet, wurden
- gehen: ging, gingst, ging, gingen, gingt, gingen
- kommen: kam, kamst, kam, kamen, kamt, kamen
- Every strong verb stem change must be individually correct — no guessing from patterns.
- Preserve all existing fields in each verb entry (word, type, audio, conjugations.presens, separable, synonyms, modal, _id, etc.).
- Maintain valid JSON throughout.

node << 'VERIFY'
const data = require('./vocabulary/core/de/verbbank.json');
const verbs = Object.entries(data).filter(([k, v]) => v._id);
const PRONOUNS = ['ich', 'du', 'er/sie/es', 'wir', 'ihr', 'sie/Sie'];
let errors = 0;

// Check 1: All verbs have preteritum
const missing = verbs.filter(([k, v]) => !(v.conjugations && v.conjugations.preteritum));
if (missing.length > 0) {
  console.log('FAIL: Verbs missing preteritum:', missing.length);
  missing.forEach(([k, v]) => console.log('  -', v._id));
  errors += missing.length;
}

// Check 2: All preteritum have all 6 pronoun keys with non-empty strings
verbs.forEach(([k, v]) => {
  if (v.conjugations && v.conjugations.preteritum) {
    const former = v.conjugations.preteritum.former;
    PRONOUNS.forEach(p => {
      if (!former[p] || typeof former[p] !== 'string' || former[p].trim() === '') {
        console.log(`FAIL: ${v._id} missing/empty pronoun "${p}"`);
        errors++;
      }
    });
    if (v.conjugations.preteritum.feature !== 'grammar_preteritum') {
      console.log(`FAIL: ${v._id} missing feature tag`);
      errors++;
    }
  }
});

// Check 3: Modal verbs have verb_type tag
const modals = ['moegen_verb', 'koennen_verb', 'muessen_verb', 'duerfen_verb', 'sollen_verb', 'wollen_verb', 'moechten_modal'];
modals.forEach(id => {
  const v = data[id];
  if (v && v.verb_type !== 'modal') {
    console.log(`FAIL: ${id} missing verb_type: modal`);
    errors++;
  }
});

// Check 4: JSON is valid (we got here, so it is)
console.log('\nTotal verbs:', verbs.length);
console.log('Verbs with preteritum:', verbs.filter(([k, v]) => v.conjugations && v.conjugations.preteritum).length);
console.log('Errors:', errors);
process.exit(errors > 0 ? 1 : 0);
