**Data table structure per entry:**

Standard haben verb:
```json
{ "participle": "gemacht", "auxiliary": "haben", "former": { "ich": "habe gemacht", "du": "hast gemacht", "er/sie/es": "hat gemacht", "wir": "haben gemacht", "ihr": "habt gemacht", "sie/Sie": "haben gemacht" }, "feature": "grammar_perfektum" }
```

Standard sein verb:
```json
{ "participle": "gegangen", "auxiliary": "sein", "former": { "ich": "bin gegangen", ... "sie/Sie": "sind gegangen" }, "feature": "grammar_perfektum" }
```

Dual-auxiliary verb (6 verbs: fahren, fliegen, schwimmen, laufen, ausziehen, wegfahren):
```json
{ "participle": "gefahren", "auxiliary": "both", "dual_auxiliary": true, "auxiliary_note": { "sein": "Bewegung ohne Objekt: ...", "haben": "Mit Objekt (transitiv): ..." }, "former": { "ich": "bin/habe gefahren", ... }, "feature": "grammar_perfektum" }
```

Modal verb (7 verbs: mögen, können, müssen, wollen, dürfen, sollen, möchten):
```json
{ "participle": "gekonnt", "auxiliary": "haben", "modal_note": "Ersatzinfinitiv: ...", "former": { "ich": "habe gekonnt", ... }, "feature": "grammar_perfektum" }
```

**Critical participle formation rules — apply strictly:**

- **Weak regular:** ge + stem + t (machen -> gemacht)
- **Strong:** ge + vowel-change + en (singen -> gesungen)
- **Mixed:** ge + consonant-change + t (bringen -> gebracht)
- **Inseparable (17 entries with inseparable: true):** NO ge- prefix (besuchen -> besucht, vergessen -> vergessen, bekommen -> bekommen)
- **Separable (19 entries with separable: true):** prefix + ge + stem-participle (aufstehen -> aufgestanden, einkaufen -> eingekauft)
- **-ieren verbs (5 entries):** NO ge- prefix (dekorieren -> dekoriert, interessieren -> interessiert, rasieren -> rasiert, konzentrieren -> konzentriert)
- **Modal (Ersatzinfinitiv):** ge + modal-stem + t for standalone (können -> gekonnt); former shows standalone form
- **Reflexive verbs:** Include reflexive pronouns (mich/dich/sich/uns/euch/sich) in former forms, matching presens/preteritum pattern

**Auxiliary conjugation patterns for former:**

haben: ich habe, du hast, er/sie/es hat, wir haben, ihr habt, sie/Sie haben
sein: ich bin, du bist, er/sie/es ist, wir sind, ihr seid, sie/Sie sind
both (dual): ich bin/habe, du bist/hast, er/sie/es ist/hat, wir sind/haben, ihr seid/habt, sie/Sie sind/haben

**Open question resolutions (from RESEARCH.md):**

1. sich_vorbereiten: Use participle "vorbereitet" (native German usage; the ge- insertion is NOT applied despite being separable — this is a known exception because bereiten → bereitet follows -ieren-like behavior)
2. haengen: Use "gehangen" (strong, intransitive) as default participle with haben auxiliary — the intransitive strong form is the standard reference
3. moechten_modal: Use participle "gemocht" (from underlying mögen), auxiliary "haben", with modal_note: "möchten hat kein eigenes Perfektum. Als Konjunktiv II von mögen wird gemocht als Partizip II verwendet. In der Praxis wird 'hat gewollt' oder 'hat mögen' benutzt."
