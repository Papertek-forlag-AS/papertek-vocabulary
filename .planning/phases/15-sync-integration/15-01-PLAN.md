**Step 1: Create scripts/validate-verbs.js** — Mirror the structure of scripts/validate-nouns.js exactly:
- Import Ajv2020 from 'ajv/dist/2020.js' (NOT default ajv import)
- Load core-word.schema.json and verb.schema.json
- Load vocabulary/core/de/verbbank.json
- Create AJV instance with { strict: false, allErrors: true }
- Add core-word schema, add verb schema
- Get validator via ajv.getSchema('https://papertek.no/schemas/vocabulary/verb.schema.json')
- Validate the verbbank; print PASS with entry count or FAIL with error count and details
- Exit 0 on pass, exit 1 on fail

**Step 2: Create scripts/fix-validation.js** — One-shot ESM script that fixes ALL 547 errors in a single pass:

For core nounbank (356 errors):
1. Update noun schema in-place: change `plural` field from `{ "type": "string" }` to `{ "type": ["string", "null"] }` in the noun.schema.json nounEntry definition (this fixes 23 plural:null errors)
2. Add `translations: {}` to every core nounbank entry that lacks a `translations` field (skip `_metadata` key; only add to entries with a `word` field). This fixes ~331 missing-translations errors.
3. Fix `leute_noun` entry: change `type` from `"substantiv (kun flertall)"` to `"noun"` (fixes 1 type error)

For core verbbank (191 errors):
4. Add `translations: {}` to every core verbbank entry that lacks a `translations` field (skip `_metadata`; only add to entries with a `word` field). This fixes ~148 missing-translations errors.
5. Fix Norwegian type values using this map: "sterkt" -> "strong", "svakt" -> "weak", "vanlig" -> "weak", "refleksiv" -> "reflexive", "verb" -> "weak". Skip entries where type is already an English value or type is "verbphrase". This fixes 42 type enum errors.

The script must:
- Read all files at start, apply fixes, write all files at end
- Log counts for each fix category (translations added, types fixed, schema updated, leute fixed)
- Use ESM imports (import/export), not CommonJS
- Use relative paths from project root (vocabulary/..., scripts/...)

**Step 3: Add validate:verbs to package.json** — Add `"validate:verbs": "node scripts/validate-verbs.js"` in the scripts section, positioning it near validate:nouns.

**Step 4: Run fix-validation.js** — Execute `node scripts/fix-validation.js` from project root.

**Step 5: Verify fixes** — Run both `npm run validate:nouns` and `npm run validate:verbs`. Both must exit 0 with zero errors.
