  <action>
Take the ~400-550 deduplicated candidates from Task 1 and apply linguistic judgment to each word. Classify every word, then write the output file.

**Classification rules — apply in this order:**

1. **Attributive test ("ein ___er Mann"):** Does the word sound grammatically correct in "ein [word]er Mann"? If NO → exclude (not an adjective). This eliminates prepositions (mit, ohne, für), pronouns (jeder, mein, dieser), conjunctions (aber, oder, wenn), adverbs that cannot be attributive (sehr, auch, schon, dort, hier, gern, fast, oft, immer, manchmal, vielleicht), particles (doch, ja, mal, denn), and interjections.

2. **Participial exclusion (per locked decision):** If the word derives from a verb — either a past participle (ge-___-t, ge-___-en patterns) or present participle (___-end pattern) — EXCLUDE it. This applies even if it passes the attributive test.
   - Known exclusions from research: geöffnet, aufregend, dringend, ausreichend, befriedigend, entspannend
   - Borderline participials that are now fully lexicalized adjectives: flag `abwesend`, `anwesend`, and `wütend` with `review: true` and a `review_note` explaining the borderline status. Do NOT exclude these — include them in the candidates list per the research recommendation.

3. **Dual-role words (adverb/adjective):** If a word can be both adjective and adverb but passes the attributive test, INCLUDE it. The existing bank already has many dual-role words (schnell, laut, leise). Only exclude words where the adverbial use is overwhelmingly dominant and the attributive use is truly ungrammatical. If borderline, include with `review: true`.

4. **Compound adjectives:** Include if they appear in the Goethe source. Do not fabricate compounds not in the source.

**For each word that passes (adjective or borderline-include):**
- `word`: the word as it appears in the Goethe list (trimmed, lowercase)
- `_id`: generate using the established pattern: lowercase + umlaut substitution (ü→ue, ö→oe, ä→ae, ß→ss) + `_adj`
- `cefr`: the CEFR level from Task 1 (already lowest-level-wins)
- `source`: the goethe-XX source tag from Task 1
- `review`: boolean — `true` only for genuinely borderline cases (participials-as-adjectives, dual-role uncertainty). Do NOT flag every word — only cases with real ambiguity.
- `review_note`: string — present only when `review: true`, explaining why it is flagged
